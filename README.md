# Lift Tracker App

The Lift Tracker App is a strength training application designed to help users follow a structured workout program, log performance data, and monitor strength progression over time[cite: 1]. The system focuses on making progressive overload efficient by replacing scattered notes and memory with reliable historical data[cite: 1].

## Core Features

* **Calendar Workout View**: A calendar interface where each day displays scheduled exercises[cite: 1].
* **Workout Logging**: Users can enter weight and repetitions completed for each exercise to create a historical record[cite: 1].
* **Previous Performance Display**: Automatically shows the most recent logged weight and reps when a lift reappears[cite: 1].
* **Program Editor**: Allows users to personalize their training structure by editing exercises, sets, and reps[cite: 1].
* **Estimated 1RM Tracking**: Calculates and displays estimated one-rep max (1RM) progression to provide measurable insight into strength growth[cite: 1].

## Technology Stack

* **Frontend**: React (Component-based structure for dashboards and calendar views)[cite: 1].
* **Backend**: Python (Handles authentication, data management, and business logic)[cite: 1].
* **Database**: SQL (Relational structure for users, exercises, and logs)[cite: 1].

## Requirements & User Stories

### Key User Stories
* **Planning**: As a lifter, I want to see my scheduled workout for today[cite: 1].
* **Tracking**: As a user, I want to log weight and reps to track progress[cite: 1].
* **Progression**: As a returning user, I want to see my previous lifts to ensure progressive overload[cite: 1].
* **Customization**: As a user, I want to edit my workout program to adjust my training plan[cite: 1].

### Technical Requirements
* **Persistence**: Data persists securely in a cloud database[cite: 1].
* **Security**: Authentication protects personal workout history, with data transmitted via HTTPS[cite: 1].
